# GenStation — Issues & Feedback

This repository is for bug reports and feature requests for [GenStation](https://genstation.art), an AI image and video generation app for macOS and iOS.

## How to Submit

- **Bug Report** — [Open a bug report](../../issues/new?template=bug_report.md)
- **Feature Request** — [Open a feature request](../../issues/new?template=feature_request.md)
- **General Question** — [Start a discussion](../../issues/new)

## Before Submitting

- Search [existing issues](../../issues) to avoid duplicates
- Include your macOS/iOS version and GenStation version
- For bugs, describe what you expected vs. what happened
- Screenshots are always helpful
